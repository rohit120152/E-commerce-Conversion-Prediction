# E-Commerce Conversion Prediction using Logistic Regression

## Overview
Built a machine learning model to predict customer conversions using demographic, campaign, and behavioral data.

## Features
- Missing Value Handling
- Label Encoding / One-Hot Encoding
- Feature Engineering
- Feature Scaling
- Logistic Regression
- Class Imbalance Handling

## Results
- Validation F1 Score: 0.56
- Private Test F1 Score: 0.53

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
